Motivation

Galaxy's domain-agnostic platform extends well beyond bioinformatics. The Digital Humanities and Social Sciences SIG connects researchers who need computational tools for analyzing large text corpora but often lack programming expertise.

We integrated six comprehensive NLP tools into Galaxy, enabling researchers to perform sophisticated text analysis without writing a single line of code. All tools are now submitted to the Galaxy Tools IUC for community integration.

NLP Tool Suite

Galaxy now includes a comprehensive ecosystem of NLP tools for multi-language text analysis:

Architecture

Each tool in Galaxy follows a consistent pattern designed for portability and ease of administration.

Galaxy Tool Wrapper
↓
Docker Container
Data Manager
↓
Language Models
Data Tables
  • Tool wrappers define inputs, outputs, and the command pipeline in Galaxy XML
  • Docker containers provide isolated, reproducible execution environments
  • Data managers download language models on demand and register them in Galaxy's data tables
  • Data tables present installed models as selectable dropdowns in the tool interface

Output Formats

All tools support multiple standard NLP output formats for interoperability:

JSON CoNLL CoNLL-U Plain Text XML (CoreNLP) GeoJSON (Geocoding) TSV (Analysis)

Structured outputs enable downstream processing with other Galaxy tools or external analysis pipelines. Enhanced output includes proper Galaxy datatypes for seamless tool chaining.

Use Cases

  • Literary analysis — extract characters and relationships from novels across languages
  • Historical research — NER on archival documents to identify people, places, and dates
  • Sentiment analysis — track emotional arcs in literary works or political discourse
  • Multilingual studies — compare NLP annotations across translations of the same text
  • Corpus linguistics — POS tagging and dependency parsing at scale

Availability

All NLP tools and data managers have been submitted to the Galaxy Tools IUC for community integration and will be available through the Galaxy Toolshed upon acceptance.

The comprehensive toolkit includes six NLP analysis tools (Stanford CoreNLP, spaCy, Stanza, co-occurrence analysis, entity geocoding, and VADER sentiment) plus three data managers for downloading language models.

For Administrators

  • Install tools from the Galaxy Toolshed
  • Use data managers (Admin → Local Data) to install only the language models your researchers need
  • No manual configuration of model files or paths required
  • CoreNLP: select languages from 8 options
  • spaCy: choose from 70+ models across 25+ languages
  • Stanza: 80+ languages with fast CPU-only models
  • Analysis tools: no model downloads required

Example Workflow: Named Entity Recognition of Nixon's Presidential Speeches

Applying Named Entity Recognition to Nixon's presidential speeches reveals clear alignment with Cold War rhetoric. As the American president, the United States of America is naturally his most often referenced location.

The other locations, Moscow and Beijing, are used as pars pro toto to stand in for Russia and China, the US's most prominent geopolitical rivals during the Cold War.

Impact: The workflow can quickly extract this information and provide geographic visualizations that guide further close reading and reveal patterns in political rhetoric.

Why Not Just Use an LLM?

  • Reproducibility — Traditional NLP tools produce identical results on every run. LLMs are non-deterministic: the same prompt can yield different entity tags or miss entities across runs. For citable, replicable research this matters.
  • Structured output — Standard NLP formats (CoNLL-U, JSON with token offsets, dependency trees, GeoJSON) are machine-parseable and consistent. LLM output is free text requiring additional parsing with no schema guarantees.
  • Scale and cost — Processing a 120-chapter novel or a 500-document corpus costs nothing beyond compute time. LLM APIs charge per token and are rate-limited, making large-scale text analysis expensive.
  • Platform synergies — Galaxy allows you to connect the NLP tools with more than 4000 tools of your choice, encompassing the whole research life cycle and not just one substep.
